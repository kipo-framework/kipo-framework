# Kipo ETL Framework

Convention over Configuration ETL Framework for Python.

## Philosophy

Kipo is a rigid but fast framework for data engineers who want to ship pipelines without spending hours on configuration.

## Features

- **Strict Structure**: Pre-defined layout for clarity and consistency.
- **Fast**: Built on top of `polars` and `uv`.
- **Beautiful CLI**: Powered by `typer` and `rich`.

## Getting Started

### Prerequisites

- [uv](https://github.com/astral-sh/uv) installed.

### Installation

```bash
uv sync
```

### Usage

```bash
uv run kipo hello
```
